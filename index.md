## 🎯 Judging Criteria (official, from openai.devpost.com)

1. **Technological Implementation** — How thoroughly and skillfully does the project use GPT-5.6? Does the code reflect genuine effort and a working, non-trivial implementation?
2. **Design** — A working/runnable project with a complete, coherent product experience — not just a technical proof of concept.
3. **Potential Impact** — A credible, specific case for solving a real problem for a real audience, actually addressed by what's demonstrated.
4. **Quality of the Idea** — A creative, non-obvious use of GPT-5.6, with genuine understanding of the problem space.
