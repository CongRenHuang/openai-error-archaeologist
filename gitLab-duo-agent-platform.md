# GitLab Duo Agent Platform

## 📊 Event Overview and Scale

> **Core Focus:** Participants utilized the [GitLab Duo Agent Platform](https://about.gitlab.com/gitlab-duo-agent-platform/) to develop AI agents capable of proactively intervening in workflows, responding to events, and executing tasks on their behalf, rather than just building Q&A chatbots.

- **Partners:** Co-sponsored and hosted by GitLab, Google Cloud, and Anthropic.
- **Event Scale:** Attracted nearly 7,000 developers who built over 600 AI agents and workflows in just a few weeks, with a total prize pool reaching $65,000.

### 🏆 Core Grand Prize Winners

- **Hackathon Grand Prize — [LORE](https://devpost.com/software/lore-living-organizational-record-engine):**
- Designed to solve the problem of "knowledge loss caused by departing engineers."
- The system features 8 agents, logic to prevent infinite loops in the knowledge graph, and carbon footprint tracking, demonstrating an exceptionally high level of completeness (including 43 tests).

- **Google Cloud Awards:**
- **Winner — [Gitdefender](https://devpost.com/software/gitdefender):** Automatically finds security vulnerabilities, writes fixes, and opens reviews within the code review workflow, requiring zero human intervention.
- **Runner-Up — [Aegis](https://devpost.com/software/aegis-2m1oq0):** Provides explanations for every decision made by the AI and is ready for production environment deployment.

- **Anthropic Awards:**
- **Winner — [GraphDev](https://devpost.com/software/graphdev):** Maps code relationships and visualizes system changes over time, allowing developers to foresee the scope of impact before making modifications.
- **Runner-Up — [DocSync](https://devpost.com/software/pipeheal):** Synchronizes documentation and code through three agents (Detector, Writer, and Reviewer), and automatically opens code reviews or creates issues.

### 🌟 Special Category Winners

- **Most Technically Impressive — [Time-Traveler](https://devpost.com/software/time-traveler-w3cxp0):** Tests database migrations by creating a secure replica of the production environment to prevent breaking real-world setups.
- **Most Impactful — [RedAgent](https://devpost.com/software/redagent):** Reviews security reports generated by other AIs, addressing the lack of trust development teams often have in AI-generated findings.
- **Easiest to Use — [Launch Control](https://devpost.com/software/launch-control-bgp8az):** Delivers an excellent user experience, solid infrastructure, and a strong sustainability score.

### 🍃 Sustainability and Green Computing (Sustainability Signal)

- **Green Agent Prize — [GreenPipe](https://devpost.com/software/greenpipe):** Scans CI/CD pipelines for their environmental impact and generates carbon footprint reports.
- **Sustainable Design Bonus:** Awarded to projects demonstrating outstanding performance in model optimization or energy-efficient architectures:
- **[BugFlow](https://devpost.com/software/bugflow-ai-regression-detective-ci-optimizer):** Turned 1 bug report into 10 fixes within 20 minutes.
- **[DELTA Cyber Reasoning](https://devpost.com/software/delta-cyber-reasoning-system):** Automated fuzz testing for security.
- **[CarbonLint](https://devpost.com/software/carbonlint):** Applied code analysis to evaluate energy usage.
- **[TFGuardian](https://devpost.com/software/tfguardian):** Features a built-in carbon footprint analyzer among other multiple agents.

### 🚀 Honorable Mentions and Future Outlook

- **Honorable Mention Projects:**
- **[SecurityMonkey](https://devpost.com/software/securitymonkey):** Injects known vulnerabilities into test branches to evaluate the capture capabilities of security scanners.
- **[stregent](https://devpost.com/software/stregent):** Monitors CI/CD pipelines, allowing developers to investigate and merge fixes directly via WhatsApp.
- **[Compliance Sentinel](https://devpost.com/software/compliance-sentinel-autonomous-devsecops-governance):** Scores the compliance risk of merge requests and automatically blocks critical violations.
- **[Carbon Tracker](https://devpost.com/software/carbon-tracker-ij25kf):** Calculates the carbon footprint of CI/CD jobs and provides optimization recommendations directly within merge requests.
- **[RepoWarden](https://devpost.com/software/docuguard):** A dynamic specification engine that records the "why" behind code creation, not just its functionality.
- **[MR Compliance Auditor](https://devpost.com/software/mr-compliance-auditor):** Collects merge request evidence, maps it to SOC 2 controls, and streams it in real-time to a compliance dashboard.
